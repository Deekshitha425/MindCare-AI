# MindCare AI

AI-powered mental health platform featuring:

- Emotion Detection (TF-IDF + LinearSVC)
- Depression Prediction (Machine Learning)
- Mental Health Chatbot
- Streamlit Dashboard
- Power BI Analytics

## Features

- Emotion classification from text
- Depression risk assessment
- Context-aware chatbot
- Analytics dashboard
- Conversation history tracking

## Tech Stack

- Python
- Streamlit
- Scikit-Learn
- XGBoost
- Plotly
- Power BI

## Datasets

This project uses publicly available Kaggle datasets.

Student Depression Dataset
